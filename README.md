# Python File Collection

This repository contains several beginner Python programs created for learning and practice.

## Files Included
- adventure_game.py – A simple text-based adventure game.
- cover_stories.py – A Python program related to story generation.
- meal_price.py – Calculates the total cost of a meal including tax and tip.
- shopping_cart.py – Simulates a basic shopping cart system.
- word_puzzle.py – A word puzzle game written in Python.
